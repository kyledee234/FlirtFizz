- **Auth**: OAuth 2.0

## 🚀 Quick Start  

1. **Clone repo**  
   ```bash  
   git clone https://github.com/yourusername/flirtfizz.git  
   ```  

2. **Install dependencies**  
   ```bash  
   cd flirtfizz  
   npm install  
   ```  

3. **Run dev server**  
   ```bash  
   npm run dev  
   ```
